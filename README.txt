สิ่งที่แก้ใน v2: "เฉพาะ" ระบบอัปโหลดทะเบียนราษฎร์ PDF
- อัปได้ 1–6 ราย (ราย 1..6)
- เว้นช่องได้
- ระบบนับจำนวนไฟล์ที่อัปจริง
- เลือก template ตามจำนวนไฟล์: template1..template6
- เรียงตามลำดับที่ผู้ใช้ส่ง (ราย 1 → 6)

วิธีรัน:
1) ติดตั้ง
   pip install flask
2) รัน
   python app.py
3) เข้า
   http://127.0.0.1:5000

หมายเหตุ:
- วางไฟล์ template ที่ templates_word/ ชื่อ template1.docx ... template6.docx
- โค้ดสร้างเอกสารจริงใน v2 นี้ทำเป็นไฟล์เปล่าเพื่อทดสอบ flow (ไม่ไปแก้อื่น)
